L2J Server GS
===

L2J Server 是一款完全以 Java 編寫的開源伺服器模擬器，適用於著名的韓國 MMORPG。
---

具有 Java JDK 17+ 和 MariaDB 10.5+ 的 Windows 或 Linux 系統。

至少 4GB RAM 和 2 個核心。

開啟連接埠 2106 和 7777。

使用 l2jserver HighFive 版本更新
使用10幾年前的 l2jtw 中文化來做編譯版本 
兩者融合成為 l2jstar 版本
